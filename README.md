# LyricTagX
Графический интерфейс для скрипта, заполняющего USLT тег, заказанный [Роман Иванов [Irimsk]](https://www.fl.ru/users/irimsk/)

## Установка
* Установите зависимости: 
```
pip install -r requirements.txt
```
## Использование
* Запустите скрипт:
```
python main.py
```
